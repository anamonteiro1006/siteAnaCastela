# 🏇Site Ana Castela

Este projeto é o meu ponto de partida no desenvolvimento de sistemas web. Trata-se de um site simples construído exclusivamente com **HTML** e **CSS**, focado em entender como a estrutura e a estilização de uma página funcionam na prática.

## 📖 O que foi praticado?

O objetivo deste repositório é documentar a evolução no aprendizado dos pilares da web:

* **HTML5:** Tags semânticas (`header`, `main`, `footer`, `section`), criação de listas, links e inserção de imagens.
* **CSS3:** Seletores, box model (margens, bordas e preenchimento), cores, fontes e alinhamentos básicos.

## 🛠️ Tecnologias Utilizadas

1.  **HTML5**: Responsável por toda a estrutura e conteúdo da página.
2.  **CSS3**: Utilizado para dar vida ao projeto, definindo o layout, as cores e a tipografia.
